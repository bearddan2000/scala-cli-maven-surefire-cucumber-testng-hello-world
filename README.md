# scala-cli-maven-surefire-cucumber-testng-hello-world

## Description
A POC for maven app using testNg
and cucumber framework with surefire plugin.

## Tech stack
- scala
- maven
  - testNg
  - surefire
  - cucumber

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
